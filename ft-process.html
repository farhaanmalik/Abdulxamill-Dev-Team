PROCESS

PRE-PRODUCTION:

  • Game Concept Analysis
  • Game Design Document
  • SRS Document
  • Project-Plan with Milestone
  • Art-Pipeline Set-up
  • Technical Design Document
  • Architecture – Design
  • Test-Plan
  • Finalising the right Art-style
  • Flow & Wireframe Doc

PRODUCTION:

  • Start developing on the gaming modules
  • Prioritise the most crucial parts and move it on to early sprints.
  • Start Creating the Art-Assets
    • Game Objects
    • Background /Environments
    • Animations
    • Visual – effects
  • Level-Design
  • UI – Elements creations
  • Game Balancing & Depth
  • Audio Creation
  • Review on each sprints & Work on the feedback .
  • Optimization & Module Testing
  • Integration of all the Modules.

POST-PRODUCTION:

  • 3rd Party SDK’s Integrations
  • QA & Testing
  • UAT
  • Final Game Deliverables
  • Monetisation
  • Appstore Submissions
  • Support & Maintenance
